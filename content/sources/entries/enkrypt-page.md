**eикrуpт** (inspired by [cryptii](https://cryptii.com/)) is a browser-only encryption/decryption tool. Everything runs in JavaScript — no backend, no data leaves your browser.

## Features

- AES, DES, and other symmetric encryption algorithms via [CryptoJS](https://cryptojs.gitbook.io/)
- Steganography: hide messages inside images via [steganography.js](https://www.peter-eigenschink.at/projects/steganographyjs/)
- Zero server involvement

## Reception

The tool [went viral on Reddit](https://www.reddit.com/r/InternetIsBeautiful/comments/3glej1/enkryptpage_encryptdecrypt_any_message_in/) after posting to r/InternetIsBeautiful.
